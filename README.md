# flutter_speech_recognizer

간단한 음성 인식 플러그인.

## v0.1.0
 - Locale 세팅.
 - 음성 인식 결과 반환.

## v1.0.0
 - IOS 지원(shake sensor 제외)
 
## 지원 OS
 - AOS
 - IOS

## 프로젝트에 적용 방법

pubspec.yaml의 dependencies에 아래부분 추가.

```
flutter_speech_recognizer:
  git:
    url: https://github.com/SugyoIn-LBSTech/flutter_speech_recognizer.git
```
